# Post #003 — Pencil démo concrète
**Date :** à planifier
**Thème :** No-code / outils IA

---

## Version finale

---

J'ai tapé une phrase. 30 secondes plus tard j'avais un site immobilier complet, design + code.

Zéro clic. Zéro Figma. Zéro CSS écrit à la main.

---

La semaine dernière je vous parlais de Pencil, l'outil qui laisse l'IA designer à ta place.
Plusieurs d'entre vous m'ont demandé à quoi ça ressemble concrètement.

Voilà exactement ce que j'ai fait avec **Pencil** dans mon éditeur de code.

Un seul prérequis : créer un fichier `.pen` vide. C'est tout.

**Étape 1 - je décris le design :**
*"Site immobilier de prestige. Header, hero avec photo, 3 fiches de biens avec image et prix, bande de stats, footer."*

Pencil génère le design visuel en direct dans l'éditeur. Header élégant, hero avec photo Unsplash, cartes de biens avec prix en italique, stats en fond noir. Tout.

**Étape 2 - je demande le code :**
*"Génère le composant React depuis ce design."*

Il produit un fichier `maison.tsx` propre, avec les vraies images, les vraies couleurs, les vraies typos du design. Je l'ouvre dans le navigateur. Le site tourne.

Deux phrases. Un site complet.

La différence avec Figma : sur Figma tu designs, puis tu passes à un dev pour coder.
Ici, les deux étapes se font dans la même conversation.

---

- De l'idée au site en production sans savoir coder
- Design + code générés dans la même conversation
- Fonctionne dans VS Code, Cursor, Claude Code

Tu veux lancer un projet sans dev ? C'est le workflow à tester en premier.
